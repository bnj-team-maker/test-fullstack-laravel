Contexte du Projet

TaskFlow Pro

TaskFlow Pro est une startup française spécialisée dans la gestion de projets et la productivité d'équipe. Vous êtes recruté comme développeur full stack pour refondre le module de gestion de tâches de leur plateforme existante.

Le système actuel présente des limitations critiques : performance dégradée, interface obsolète, pas d'API mobile, et difficultés de scalabilité. Votre mission est de développer un nouveau module moderne qui améliorera la productivité de 60% et supportera 10 000 utilisateurs simultanés.

Guide d'Installation

1

Créer le Projet Laravel

composer create-project laravel/laravel:^10.0 task-manager
2

Installer les Dépendances

composer install
npm install
npm install @inertiajs/vue3 vue@^3.0.0 @vitejs/plugin-vue
npm install @tailwindcss/forms ziggy-js
3

Installer Breeze et Livewire

composer require laravel/breeze --dev
php artisan breeze:install vue
composer require livewire/livewire:^3.0
4

Configuration

cp .env.example .env
php artisan key:generate
# Configurer la base de données dans .env
php artisan migrate --seed
5

Démarrer les Serveurs

# Terminal 1
php artisan serve

# Terminal 2
npm run dev

Exemples de Code

Modèle Task 

class Task extends Model
{
    protected $fillable = [
        'user_id', 'title', 'description',
        'category_id', 'priority', 'status',
        'due_date', 'completed_at'
    ];

    public function user(): BelongsTo
    {
        return $this->belongsTo(User::class);
    }

    public function scopePending($query)
    {
        return $query->where('status', 'pending');
    }
}

Service TaskService 

class TaskService
{
    public function createTask(array $data, User $user): Task
    {
        return DB::transaction(function () use ($data, $user) {
            return Task::create([
                'user_id' => $user->id,
                'title' => $data['title'],
                // ...
            ]);
        });
    }
}
